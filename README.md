
# Youtube Converter

A simple Youtube converter to download videos in MP3 or MP4. Build Using Nextjs and [Shadcn/ui](https://ui.shadcn.com/)


## Installation

Clone the repository and run NPM install.
```bash
  npm install ytconverter
```
